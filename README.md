# webserver

Gin 을 이용한 웹서버 구현
- Gin is a web framework written in Go (Golang).
- See [Gin Web Framework](https://github.com/gin-gonic/gin/) for more information.

## Getting Started
### 1. Install Gin
 ```
 go get -u github.com/gin-gonic/gin
 ```
### 2. Build
```
go build -o vote ./server/

or

make vote
```
### 3. Run
```
./vote
```
